Use this file for `plan.md` Task 1. Fill one row per draft before editing or publishing any article.

## Audit Rules

- A **publish candidate** is only eligible for a later edit pass; it is not published during the audit.
- A **transcript-backed post** must add a distinct article thesis beyond the matching resource summary. If it mostly restates one summary, keep it draft/private and promote the summary instead.
- **Merge** means naming the target article and the exact sections/ideas to carry over.
- **Retire** means keep `draft: true`, avoid production links, and record why. Do not delete unless explicitly chosen later.
- **Keep private** means no production links and no roadmap pressure.
- Amp-specific claims must be verified close to publication and recorded as “verified as of YYYY-MM-DD” with the source used.
